## Summary & Key Takeaways

- Datasette 1.0a34 introduces new tools for managing data directly in the interface.
- Users can now insert, edit, and delete rows on table pages.
- Row-specific actions are also available on individual row pages.
- The feature was inspired by the ability to edit data via Datasette Agent's chat interface.
- This update significantly enhances Datasette's utility as a data exploration and management tool.

## Our Commentary

This is a big one for Datasette users. We've always loved it for exploration, but the lack of direct editing was a bit of a friction point. It's almost funny that the AI agent got write access before the main UI! This update makes Datasette even more powerful. It feels like it's maturing into a full-fledged data workbench.
