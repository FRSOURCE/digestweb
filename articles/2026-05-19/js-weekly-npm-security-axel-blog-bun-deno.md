## Summary & Key Takeaways

- An RFC proposes making npm install scripts opt-in for security.
- Dr. Axel Rauschmayer's blog is offline due to AI crawlers.
- Bun's Rust-based rewrite has been merged.
- Deno 2.8 is teasing significant Node.js compatibility and TypeScript 6.0.3.
- Chrome/Edge are working on a new `<install>` HTML element for PWAs.
- Express.js has a new look and improved docs.

## Our Commentary
