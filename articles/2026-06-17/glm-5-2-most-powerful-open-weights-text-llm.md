## Summary & Key Takeaways

- Z.ai released GLM-5.2, a 753B parameter open-weights LLM under an MIT license.
- It features a massive 1 million token context window, a significant increase from its predecessor.
- GLM-5.2 leads the Artificial Analysis Intelligence Index for open-weights models.
- It ranks second on the Code Arena WebDev leaderboard, despite being text-only.
- The model is available via OpenRouter at competitive pricing compared to proprietary models.
- It uses more output tokens per task than other leading open-weights models.
