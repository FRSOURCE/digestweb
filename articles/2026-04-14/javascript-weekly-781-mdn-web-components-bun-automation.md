## Summary & Key Takeaways

- MDN has completely rebuilt its frontend, transitioning from React to web components and a custom server component system to reduce shipped JavaScript.
- Phaser 4.0, the 2D WebGL and Canvas game framework, has been released with performance improvements and AI agent support.
- Bun v1.3.12 now includes native, headless browser automation and an in-process task scheduler (`Bun.cron`).
- Google plans to penalize websites that interfere with browser back button functionality starting in June.
- Critical security fixes for a React Server Components vulnerability have been released in React 19.2.5, 19.1.6, and 19.0.5.
- React Native 0.85 brings a new animation backend and devtools improvements.

## Our Commentary

This JavaScript Weekly issue is packed! The MDN news is a huge statement – ditching React for web components on such a foundational resource is a bold move that speaks volumes about the maturity and capabilities of native web technologies. Bun's continued rapid development, especially with native browser automation, is also incredibly exciting and positions it as a serious contender in the runtime space. And of course, a React security fix is always something to pay attention to. It feels like the web platform is truly coming into its own, offering compelling alternatives to traditional frameworks.
