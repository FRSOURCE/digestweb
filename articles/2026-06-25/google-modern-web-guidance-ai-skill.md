## Summary & Key Takeaways

- Google released an AI "skill" named "Modern Web Guidance" at I/O.
- It consists of nested Markdown files containing web development information.
- AI agents are designed to read and utilize this content within their context window.
- The skill aims to guide AI in performing web development tasks more effectively.

## Our Commentary

This is a smart move from Google. Instead of just hoping AI gets web dev right, they're actively curating a knowledge base for it. It's a pragmatic approach to improving AI's utility in our field. I wonder how comprehensive it is and if it will evolve with the web platform.
