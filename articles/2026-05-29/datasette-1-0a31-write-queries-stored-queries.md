## Summary & Key Takeaways

- Datasette 1.0a31 is a significant alpha release with new features.
- Users can now execute write queries against their databases.
- The release also introduces the ability to save stored queries.
- Stored queries can be private or shared within a Datasette instance.
- This expands Datasette's functionality beyond read-only data exploration.

## Our Commentary

This is a big step for Datasette. We've always appreciated its read-only data exploration, but adding write capabilities and stored queries really changes the game. It feels like it's evolving into a more comprehensive data management tool. I'm curious to see how this impacts its adoption.
