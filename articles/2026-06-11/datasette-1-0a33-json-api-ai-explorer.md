## Summary & Key Takeaways

- Datasette 1.0a33 extends the `?_extra=` API pattern to cover queries and rows.
- This pattern, previously for tables, is now fully documented.
- The release marks a significant step towards the stable 1.0 version of Datasette.
- An API explorer for the new feature was built with assistance from Claude Fable 5 and GPT-5.5 xhigh.

## Our Commentary

Datasette continues its steady march towards 1.0, and extending the `_extra` pattern is a solid step. What really caught my eye, though, is Simon's casual mention of using Claude Fable and GPT-5.5 to build the API explorer. It's a perfect, understated example of how AI is already integrating into developer workflows for practical, everyday tasks.
