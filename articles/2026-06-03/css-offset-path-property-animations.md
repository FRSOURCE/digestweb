## Summary & Key Takeaways

- The `offset-path` CSS property defines a movement path for animated elements.
- It allows elements to follow custom trajectories during animations.
- This property was previously known as `motion-path` in earlier specifications.
- All related `motion-*` properties are being renamed to `offset-*` in the CSS spec.
- It's part of the broader CSS Motion Path module.
